Hey there! You made it to the sample overlay. This template has all you need to start making a browser overlay/utility for Spooder.
Here's what's included in the bundle:

sendOSC(address, content) - You can call this anywhere in your overlay
getOSCMessage(message) - *Required* Write this function to make all your OSC triggers
onConnect() - *Optional* This function triggers when the overlay receives /connect/success from the Spooder
isExternal:boolean - This will tell whether the overlay is accessed locally or through the internet (e.g. Ngrok)
