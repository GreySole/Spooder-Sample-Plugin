Hey there! You made it to the sample overlay. This template has all you need to start making a browser overlay for Spooder.
Here's what's included in the bundle:

sendOSC(address, content) - You can call this anywhere in your overlay
getOSCMessage(message) - *Required* Write this function to make all your OSC triggers
onOSCOpen() - *Optional* This function triggers when the OSC module opens.
then write a trigger for '/sampleplugin/connect/success' in the getOSCMessage function
